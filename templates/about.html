🌌 About OrbitalTrack

OrbitalTrack is a real-time aerospace analytics dashboard built for enthusiasts, students, engineers, and recruiters.

🚀 Our Mission

To deliver interactive, real-time visualizations and educational insights from the space industry. This platform demonstrates what a well-designed, data-driven dashboard can achieve using modern technologies.

🧠 Technologies Used

  • Flask – Python microframework for backend routing and templates
  • Chart.js – Beautiful, responsive charts for launch analytics
  • Leaflet.js – Interactive global map for launch locations
  • Bootstrap 5 – Frontend UI framework for responsiveness
  • Particles.js – Visual background animations
  • Launch Library 2 API – Live space mission data feed

📊 What This Demonstrates

  • Fluent use of REST APIs and JSON data transformation
  • UI/UX design aligned with aerospace/defense industry standards
  • Responsive, reusable components (modals, filters, maps, charts)
  • Clear documentation and structured frontend/backend separation
